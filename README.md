# Token-watch
Watch new tokens on the blockchain. Currently only for ETH. TODO: BSC.
API calls takes a long time, approximately 30 seconds for 500 blocks.

## Install
1.1 Go to main directory<br>
1.2 Run ```flask run```

## Requirements
```
etherscan==0.3.0 <br>
etherscan_python==2.1.0 <br>
Flask==2.2.2 <br>
web3==5.30.0 <br>
```
